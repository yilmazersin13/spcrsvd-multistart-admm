# Data and Code Availability

The following text can be used as the *Data and Code Availability* statement in
the paper.

---

**Data and Code Availability.**
The code that implements the SPCRsvd multi-start ADMM estimator and reproduces
all simulations and figures in this paper is publicly available at
https://github.com/USERNAME/spcrsvd-multistart-admm. The two real datasets are
openly available: the diabetes dataset is distributed with the scikit-learn
library (Efron et al., 2004), and the gasoline near-infrared dataset (Kalivas,
1997) is available in the `pls` R package and from public mirrors; the
repository downloads it automatically on first use. No new data were generated
in this study.
